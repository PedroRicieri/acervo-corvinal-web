Este é o repositório do **Acervo Corvinal**, um sistema de biblioteca virtual desenvolvido como Trabalho Prático 1 para a disciplina de Programação Web II do curso de Análise e Desenvolvimento de Sistemas. O projeto apresenta uma interface temática, responsiva e dinâmica.

## ✨ Funcionalidades

* **Página Inicial Dinâmica:** Apresentação do acervo utilizando um carrossel horizontal de rolagem suave.
* **Catálogo de Livros e Autores:** Páginas dedicadas listando todas as obras e escritores em formato de *cards* modernos (UI/UX).
* **Perfis Detalhados:** Rotas dinâmicas que exibem informações específicas de cada livro (sinopse, ano) e autor (biografia e obras relacionadas).
* **Sistema de Busca Global:** Barra de pesquisa funcional no cabeçalho que filtra em tempo real livros e autores pelo nome, tratando resultados vazios ou nulos.
* **Design Temático:** Identidade visual inspirada na casa Corvinal (Harry Potter), com paleta de cores focada em usabilidade e redução de fadiga visual.

## 🛠️ Tecnologias Utilizadas

* **Backend:** C# com ASP.NET Core MVC
* **Frontend:** Razor Pages (`.cshtml`), HTML5, CSS3
* **Estilização:** Bootstrap 5 + CSS Customizado (Variáveis de cores e Flexbox)
* **Arquitetura:** Padrão MVC (Model-View-Controller) com dados mockados em memória.
